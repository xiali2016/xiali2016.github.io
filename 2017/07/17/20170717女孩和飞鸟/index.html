女孩和飞鸟

By 夏小洵

发表于 2017-07-17

丘陵圆鼓鼓，连绵起伏；
海比天更蓝，更深沉。
白帆海面生——
我和你大呼大叫，太神奇！

我喜欢玩打仗，
你中意跳皮筋。
玩翻绳游戏，总是你赢。
不知你发现没？
是我偷偷让着你。

上山拔猪菜，
拉着小手并立在山岗，
我望着远方发呆……

你关切地问：
“怎么看着不开心？
有啥心事？
是不是又挨揍了？”
我若无其事地答：
“没啥，只是喜欢瞎想，
自己多想变成一只飞鸟。”

那个村庄很小，
别人描述的外部世界很大，
我的心更大。

季节轮回，飞鸟归去，
我仍在外面的城市漂泊。
你又在哪里？
“听说”这个词
是一个忧伤故事的开头。

（2017.07.17）
