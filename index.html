十年一行诗

By 夏小洵

发表于 2021-03-14

暮春，是最美好的季节吧？
周末放下一切，走在环形小路上
身边花草树木的变化，或细微，或明显

春风清劲，从南刮来
远方的海洋讯息带回来了吗？

河水中的鱼儿游动，忙着觅食
还是寻觅一场浪漫爱情？

木棉的红花扑哧一声
重重落于银边山菅兰的草丛里
一大片落花依然像开在枝头那样显眼

勒杜鹃开在护坡的半腰
四季艳丽，如此招摇
会更引人关注，还是会落得厌烦和冷落？

知道了噪鹃的叫声分类和品种
是否意味着可以拥有它们
成为它们的统治者？

写下文章的题目：《十年一行诗》
写下开始的日子：2021.03.15

（2021.03.14）

与C姑娘的对话

By 夏小洵

发表于 2021-03-10

C：路边的樱花开了，白的像雪一样，骑车从樱花下穿过时，心有雀跃。想和你分享下这种感觉。

X：想象着你说的情景应该很美，只有经历了严冬的北方才能更深体会到春天的惊喜吧， 这里冬天也有花开，现在木棉花、三角梅、毛杜鹃和火焰树的红色花朵在撞色， 快进入潮湿而闷热的梅雨季了！

C：南方没有春天，不过四季如春。确实像你说的，经历过冬天的萧瑟，到了春天，才能体会到，万物生。

X：我妈说老家种的樱桃树，花开时也和城市的樱花一样漂亮，可我一次也没见过， 他们也从没拍照片发我，总是忙着营生，那些花对于他们是司空见惯的事物。

C：是啊，蒋一谈的截句里，就有一句：花匠的家里没有花。

（2021.03.10）

二月的芒果树

By 夏小洵

发表于 2021-02-25

二月的芒果树开花了

我写下来上面这一句

就再也写不下去了

把纸和笔放进抽屉

早上阴着天，飘着小雨

三岁的小女孩骑着滑板车

“你没有给我挡住雨呀！”

她不满地大声叫喊

旁边的爷爷躬身打着一把大雨伞

可爱的小公主啊

希望多年以后—

你成年以后，壮年以后，年老以后

还能想起这一幕

想起你曾天真无忌的叫喊

想起爷爷和蔼的，唯唯诺诺的

应答

留下几批

然然正在进行线上期中考试，
十一点微信电话还要开周一的公司例会。

我在闲鱼一边搜索二手大行P8，
一边想象着一场自行车旅行。

儿歌

我的大学，我的青春岁月

By 夏小洵

发表于 2020-01-19

玉臂
大一诗词欣赏课
女老师讲到“清辉玉臂寒”
语调顿时激动起来

玉手
跟你学转笔技巧
看不够那只玉手
想象中触碰过它，微凉

匾额
每次走近一号楼
总看到“志在蓝天”的匾额
多少年过去了，依然在心里

大学宿舍
十二吋黑白电视屏里有远方
酷热的夏天，杨柳依依，夏蝉鸣鸣
室外景物镀上一层耀眼的亮白
传达室窗口的电话拨号盘几个号码
总过多承受着青涩岁月的磨损

中学同学
白色连衣裙开满蓝色小碎花
颐和园、圆明园和北海公园
她跳下了十七孔桥的最后几阶台阶
她说：“北海公园荷花前的我，
是人生中最美的样子。”

大学毕业照
后面倒数第二排偏中间找到了你
认出你不太容易—
身形单薄瘦长，桀骜不驯的头发有点乱，廉价宽框大眼镜
面部表情严肃，望着远方，眼神冷峻
你有很多远大抱负，你是个刻苦上进的好青年

温暖
乘375路从北宫门返校
脚刚一离地就被人潮涌进车里
人和人挨得那么近，孤独的学子体验到一种温暖

（2020.01.19）

十一月末

By 夏小洵

发表于 2019-11-30

北方下了第一场雪
南方榄仁树、大花紫薇的叶子红了
簕杜鹃和大腹木棉花开正艳
本地木棉的果子熟透炸开，释放出纷纷扰扰的孩子
多像濒临崩溃的母亲终于懂得放手
夏小洵见信如晤—我正在读侯老师写的信和寄来的书
呼兰河是一座小城，每个人心中有一座小城
脚前是从花叶艳山姜，绿叶中布满白色的纹路
一张张小丑刻意画出的脸谱
在讥讽我脖子上默默蔓延的白癜风吗？
骑儿童车的小男孩哭叫着，非要妈妈宣布他才是第一
毛豆花生，烤羊肉串和羊腰，烤生蚝，再加上三扎青岛黄啤酒
明天还想去看一场电影：
听说《海上钢琴师》不错啊！

（2019.11.30）

立夏之后

By 夏小洵

发表于 2019-05-12

周末出太阳了
结束了连续几周的阴雨天
早上坐在书桌前，边听声音
边望着南窗小区里的绿树和草地
马路边的凤凰木又开花了 ——
火势还没有燃烧起来，要等些日子
夏蝉又开始欢叫
晚上换成蛙声
一年又一年，这一年又不同
新的生命出生，老去的生命走远
有些厌恶南方的夏天
它真正来临时，竟有些惊喜
苦难，悬而未决时让人担心
真正来到时，专注安心承受就好了

（2019.05.12）

诗

雨夜

By 夏小洵

发表于 2019-05-06

窗外雨声密集
雨点打在草地，叶子和地砖上
想给某人写信
想写紫檀树又开花了
树下林荫路成了条黄金大道
不写想念和爱的字眼
想写负重爬行的蜗牛
爬上湿滑的石墙，爬过石材路面
有多少会被漫不经心的漫步者碾碎？
为什么不回到最初的森林?
回到北方，回到南方
回到梦中的地方
