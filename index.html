十一月末

By 夏小洵

发表于 2019-11-30

北方下了第一场雪
南方榄仁树、大花紫薇的叶子红了
簕杜鹃和大腹木棉花开正艳
本地木棉的果子熟透炸开，释放出纷纷扰扰的孩子
多像濒临崩溃的母亲终于懂得放手
夏小洵见信如晤—我正在读侯老师写的信和寄来的书
呼兰河是一座小城，每个人心中有一座小城
脚前是从花叶艳山姜，绿叶中布满白色的纹路
一张张小丑刻意画出的脸谱
在讥讽我脖子上默默蔓延的白癜风吗？
骑儿童车的小男孩哭叫着，非要妈妈宣布他才是第一
毛豆花生，烤羊肉串和羊腰，烤生蚝，再加上三扎青岛黄啤酒
明天还想去看一场电影：
听说《海上钢琴师》不错啊！

（2019.11.30）

写在玉龙雪山下

By 夏小洵

发表于 2019-08-12

古城夜晚的灯光比日月还明亮
束河却在白天沉睡
雪山冷冽的水静静流着
五花石经历了千百年的马蹄和鞋子
泥土，花香，茶香和雪雨
鸟儿依然在绿树上歌唱
核桃在结青色果子
青砖青瓦松木柱的庭院里阳光浓烈
蜻蜓振动金色的翅膀
阳台上慵懒的旅人快醒来吧
这时刻已经到了
旅程的终点在一个遥远又繁忙的城市

（2019.08.12）

立夏之后

By 夏小洵

发表于 2019-05-12

周末出太阳了
结束了连续几周的阴雨天
早上坐在书桌前，边听声音
边望着南窗小区里的绿树和草地
马路边的凤凰木又开花了 ——
火势还没有燃烧起来，要等些日子
夏蝉又开始欢叫
晚上换成蛙声
一年又一年，这一年又不同
新的生命出生，老去的生命走远
有些厌恶南方的夏天
它真正来临时，竟有些惊喜
苦难，悬而未决时让人担心
真正来到时，专注安心承受就好了

（2019.05.12）

雨夜

By 夏小洵

发表于 2019-05-06

窗外雨声密集
雨点打在草地，叶子和地砖上
想给某人写信
想写紫檀树又开花了
树下林荫路成了条黄金大道
不写想念和爱的字眼
想写负重爬行的蜗牛
爬上湿滑的石墙，爬过石材路面
有多少会被漫不经心的漫步者碾碎？
为什么不回到最初的森林?
回到北方，回到南方
回到梦中的地方

（2019.05.06）

路过的夏天

By 夏小洵

发表于 2018-08-10

月亮藏到云后，云朵一片光亮
香蒲默默伸出棕褐色花柱
蛙鸣、虫鸣和蝉声，在等一轮预测的血月
蜗牛爬过草地，穿过路面
拖出短暂的湿痕，留下被踩坏的碎壳
壁虎钻入灯箱捕虫，一个个黑色剪影
荔枝熟透了，湿重空气中飘荡着酒香
琴叶榕结满青杏样的果子
黑紫色浆果从不知名的树上掉落
小叶榄仁的叶影投向半黑半亮的地砖
有人想着南方曾经的好天气
有人念着北方美人的容颜
小院里种了几株黄瓜
窗外烈阳之下，金灿灿的花朵闪耀

（2018.08.10）

爸爸和我

By 夏小洵

发表于 2018-05-10

我和爸爸之间的话不多，
一年也就几句吧。
爸爸喜欢窝在老家，
外面哪儿也不去。
回老家的时候，
他默默地走在前面，
带我去爷爷奶奶的坟地。

小时候爸爸拿筷子抽我脸的仇恨，
我早释然了。
当然，我也继承了爸爸的好品质，
脾气发作时，我会拿皮带抽儿子的脸和背……
打孩子是不对的，我知道的，
但是做爸爸，做人，需要一种快感。
就像在这世界上你经常会遇到些傻X，
表面上你不动声色，
内心里会忍不住痛快地问候一个字：操！

（2018.05.10）

三行诗（14）

By 夏小洵

发表于 2018-04-02

四月
最短的二月
连接着最漫长的三月
最美是人间的四月天

小狗
养一只小狗
陪它慢慢长大
昨日重现

清明
雨雪，鸟鸣，春花
忙碌的人慰问安息的人
苟活者追思永生者

打架
男人打架还能为了什么？
不是为争地盘
就是为争女人

母子
厌恶自己的妈妈
逃离，选择孤独的远方
最终又娶了一个像妈妈的女人

倒春寒
樱花开满山坡
一夜寒风吻过
冷酷之爱不会结果

电话
爸爸说：樱桃树开花了
妈妈说：谷雨时把黄瓜苗移出里屋
我说：在魔都一切都好

三行诗（13）

By 夏小洵

发表于 2018-03-29

花期
三月的阳光和暖风多美好
为何惴惴不安，心神不宁？
错过了花期，错过了有关你的信息
缺陷
谁用想象力修补我的缺陷
那些令我自卑的病症
如木板花纹闪着迷人的光彩
午后
晌午，饱餐之后昏昏欲睡
车辆的轰鸣沉闷，鸟雀的鸣叫清脆
碗碟在碰撞，妈妈在呵斥孩子
春风
我在南方
妈妈在北方
春天的风，从南方刮向北方
春夜
夜风清劲
女孩在十字路口唱着忧伤情歌
“为母治病”几个字歪歪扭扭
碰撞
女人柔软的情感
男人坚硬的逻辑
碰撞，碾压，糅合
春光
百叶窗切割春光
沙发布面上光影倾斜
我们存在于这尘世，只是一刹那
海风
面朝大海，海风吹拂
那时你偎依在我怀里
是否还记得说过的情话?
玉手
跟你学转笔技巧
其实是看不够那只玉手
只在想象中触碰过它，微凉

（2018.03.29）

春日

By 夏小洵

发表于 2018-03-14

冰冷的暗夜过后，清晨窗外雄鸟的鸣叫太夸张
隔壁女婴咿咿呀呀努力去回应
翻掏小挎包重新找回一张购物卡

黄花风铃木金色的花朵，湛蓝天空下闪光
木棉花火红厚重，微醺的春风里不时掉落
希冀和绝望的心，温暖和冷漠的心，戴着相似面具
文字的河流，载着记忆和迷失，或安静，或湍急

为几棵开花的树，在朋友圈询问，确认名字
主播感冒了，嗓音沙哑，别有韵味
在纽约的寒风中行走，路过建筑、街景和看不懂的画作
秀出精致的早餐和晚餐
感慨春花那么快凋谢

几年前的虚拟记忆，身体内部形成了实体构造
春天来了，交配季节到了，让更多的狗到来吧

（2018.03.14）
