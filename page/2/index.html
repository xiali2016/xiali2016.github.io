可以怎样开始？

By 夏小洵

发表于 2018-01-14

可以怎样开始？

可以从一个微笑开始
从一个眼神相对的片刻沉默开始

可以从偷偷的喜欢开始
从若有若无的憎恨开始

可以从一个玩笑开始
从一个表情符号开始

可从一段文字开始
从一张照片开始

可以从一句怯生生的语音开始
从一首动听的歌儿开始

可以从一个酣睡初醒的午后开始
从一个无眠的深夜开始

可以怎样开始？

（2018.01.14）

陌生人 | 聊骚

By 夏小洵

发表于 2017-12-24

陌生人
不问来自哪里
不问去往哪里
缠绵之后，是否孤独依旧？
相互叮嘱：走路一定要小心吖
不要跌进那口黑漆漆的深井

陌生人
假面舞会从清晨，还是从午夜开始？
内心之门已经全敞开
要拿走什么，会留下什么？
不要着急，也不必惊慌

陌生人
带来镣铐、绳索和皮鞭，解救
拿出矜持、喘息和汹涌的潮水，回应
什么是绝对命令，什么是完全臣服？
踏进心底的深处吧
是光明、黑暗，还是灰色的模糊？

陌生人
唤醒几百万年前的两头野兽
不要撕裂赤裸的胸膛
刺入心中的每根利爪
都会一一拔出
再寻机去刺入别人的心脏

（2017.12.24）

一天的作息

By 夏小洵

发表于 2017-12-17

我这样安排一天的作息：
清晨，到湖边把有关你的一切
悄悄投进幽暗的湖底
中午，走很远的路
去那家曾一起吃过饭的小餐馆，点份套餐
傍晚，我又独坐在起风的湖边
吸烟，思念

你知道吗？
黄昏到来的时候
田里的花生秧会闭合对称的叶片
路旁的葱兰会收拢洁白的花瓣

夜空的尽头是寒星，还是虚无的黑暗？
语言的尽头是沉默，还是心中萦绕的歌声？

我不能写信给你
想对你说的每句话
都是些甜蜜又苦涩的秘密
我多么担心：这些话语一旦泄露
终将变成荒谬的玩笑

（2017.12.17）

明月
岸边人潮人海
等待明月升起
月是故乡明，还是外国的月亮更圆?

纸飞机
童年的纸飞机飞到哪儿了？
还会飞回你我的手中吗？
这里起风了，你那里呢？

苍耳
悄悄把两个苍耳粘到你的小辫子上
喜欢看你嗔怒时红扑扑的脸
一直没敢对你说，那时你好可爱

山东

北京路不在北京
山东街不在家乡

《黑骏马》

她的身体已是饱胀的花骨朵
含苞欲放，我也长大了

烈酒
烈酒沉入胃里
身体漂浮起来
灵魂和肉体若即若离

签名
一个姑娘的签名：想做
思想上的女流氓，生活上的好姑娘
外形上的柔情少女，心理上的变形金刚

（2017.10.07）

《童年的风》童谣

三行诗（10）

By 夏小洵

发表于 2017-09-28

葡萄
十月快到了
家乡的葡萄熟了
一想，嗓口齁甜齁甜

陪伴
一直陪在一个人身边
说来容易，做到挺难
一切随缘最好

喵酱
跟着喵酱读《哈利波特》
读孩子的书
听孩子的声音

心境
心情好的时候
天空的蓝色清澈透明
几朵孤独的流云得到收留

美景
景色的美
在于其带来的忧伤
想到了黛玉葬花

红蜻蜓
小女孩：“妈妈，那条红色蜻蜓真漂亮！”
妈妈：“应该是那只红蜻蜓呀，
鱼是一条一条的，蜻蜓要用‘只’。”

小狗
家里的小狗过生日
老豆老调重弹
“老乡孔子也今天生日，加油孩子！”

哑巴石
走东山的小路去庄稼地
脚心踩到块石头，右脚肿疼了老多天
那是块哑巴石，当时应该“哎呀”大叫一声的

（2016.09.28）

诗雪花体诗

三行诗（09）

By 夏小洵

发表于 2017-09-21

余秀华的诗
摇摇晃晃的人间
月光落在左手上
我们爱过又忘记

秋雨
一场秋雨一场凉
天空落泪一次，热情减退一次
她需要的是发泄，不是爱情

对抗
选择了对抗
也失去了自由
放下，才能解脱

生日
小女孩：“ 为什么有人能过三个生日？”
妈妈：“只有两个噢，阳历的和阴历的。”
小女孩：“妈妈，我就是要过三个生日！”

午后
午后的时光很恍惚
一个少年梦到已经长大
一个老人梦见回到少年

家长会
一年一度的家长会
孩子们长大了
爸爸妈妈们仍没习惯老去

三行诗
对我来说
就是写三行字
不多，不少

钉子户
做一颗永不生锈的螺丝钉
你做到了啊
但不在当权者指示的位置

积极心态
学习有关幸福的课程
发现了天朝强盛和西方衰落的秘密
他们的报导多消极，我们的宣传多正面

人性的恶
多了解人性的恶
不为了报复
而为了宽恕

高考
后来，那噩梦做了多少次
功课还没有复习
明天就要参加考试

（2017.09.21）

九·一八
拉响刺耳的警笛
回到那个悲惨时代
凶恶的豺狼闯进我家乡

陈粒的歌
“我想要更好更圆的月亮” 【1】
记得第一次听你的歌
不记得最后一次对你说的话

蝙蝠
世界的盲者
看不见太阳的光明
听得到黑暗的自由

秋虫
黑夜降临
秋虫合鸣
远古的夜晚也这样么？

倒影
湖面上灯光的倒影被拉长
摇曳，断断续续
影子比实物的本身更美，更诗意

决定
我决定不再想你了
明年的桃花还会绽开
那不是我能控制的事情

沉重
车流的发动机轰鸣
小区楼房的装修电钻声
让夜晚的梦更沉重

筱暖
十二小时的时差
一个太平洋的距离
熟悉和陌生之间的平衡刚刚好

绵阳
大山，稻田，清澈的溪水
一个叫安县的地方很安静
城里的姑娘很漂亮

接受
接受人生的苦难
接受自己的不完美
接受爱

毒品
我要戒除这些毒品：
烟，酒，可乐
对你无休止的思念

（2017.09.20）

不该在十七岁爱上一个人

By 夏小洵

发表于 2017-09-17

十二吋黑白电视屏有远方的夏天
杨柳依依，夏蝉鸣鸣
室外景物镀上一层耀眼的亮白

电话拨号盘的几个号码
一直承受青涩岁月的磨损
突如其来的一场真挚
无法控制的一段冲动

白色连衣裙开满蓝色小碎花
不是所有夏花都心想事成地绽放
不是所有花朵都在秋天结出果子

谋划周密的私奔，注定
折断浪漫的翅膀，跌回现实的俗套
后来玩真心话大冒险，她说
“不该在十七岁爱上一个人”

微笑吧，忍住泪水
投入地唱完这首歌
十年后的诺言谁还记得？
歌词中的几句多像在写曾经的我们

（2017.09.17）

三行诗（07）

By 夏小洵

发表于 2017-09-13

遇见
当初遇见你在哪个日子?
你问我还记得第一次吗？
我反复揣摩问话的含意

着急
我不再着急
比如，慢慢看完这本书
而不是一次购进全集

眼睛
你的右手臂支起，捂住右眼
只留另一只桃花眼，半睁半闭
神情迷离，像问候，又像告别

容器
心是神奇的容器
装得下灰色的石头
白色的骨头

草帽
喜欢戴顶白色的草帽
慈祥地望着你
那时我多像你的父亲

歌声
这是同一首歌啊
为何有人听后微笑
有人听后流泪？

伤害
他的利箭射穿你的心
你拔出来，又刺入我的心脏
一种快感，一种补偿

心碎
心碎是什么感觉？
我没有问你
你也不必问我

九月
九月来了
最美好的季节到了
也是我们该告别的时刻了

删除
删除一个联系人
删不了某段记忆
删不了某些午后时光

无声
无声的告别
是深情的
也是绝情的

明天
从明天起做个幸福的人
不断重生的人
不断死去

（2017.09.13）

三行诗（06）

By 夏小洵

发表于 2017-08-26

大火
炙烈的大火
燃烧得短暂，熄灭得猝然
担心我们之间的爱也会这样

魔鬼和神
不再惧怕黑暗，黑色的事物
亲近更像自己的魔鬼
畏惧一本正经的神

云朵
云朵在微风中气定神闲，不紧不慢
即将离开视线的一瞬间，倏然加速
像极了我的中年岁月

痛苦
树折断一根枝丫
人失恋一次
哪个更痛苦？

断臂
台风天鸽刮过
大树成了断臂的维纳斯
被蹂躏强暴后，如何面对晴朗的天空？

漩涡
台风的巨大漩涡掠过树林
树木舞动着无数只手臂
是兴奋，恐惧，还是身不由己？

邮箱
静静地待在少为人知的角落
遇见是种缘分
记住要给我写信

懂得
不奢求、不忍心让你懂得我
我的这些苟且啊
到了中年才能体会

写诗
我在小米笔记本上写字时
身体轻盈，是蓝天上飘来飘去的云
月结单短信又让这身体重重摔回地面

（2017.08.26）

诗三行诗

台风天鸽

By 夏小洵

发表于 2017-08-25

一只巨大的鸽子飞过天空
清洗世界表层的肮脏不堪
雾霾，尘埃，碎纸，塑料袋
山寨的天堂伞，病因可疑的失语症

台风洗净污浊，去除不可靠的构筑物
也暗暗摧毁自己不喜欢的：
椰子树的一条灰色的大叶子
凤凰木的两根被虫蛀的枝丫

天空的蓝恢复到起初的蓝
蓝得总想看，又不忍多看
蓝得触手可及，蓝得心中不安
隐匿的心事是否还安全、保险？

这些天，云层淋漓极致地展现了两面
魔鬼的面目：
巨大的能量漩涡，黑暗，狂野，破坏
天使的脸庞：
连绵的白色的梦，纯洁，天真，温情

万寿无疆的红太阳又一次升起
拯救了广袤的、饱受摧残的大地
我依然低调，轻声宣布：
批准这个星球的天朝国正式入秋！

（2017.08.25）

梦

By 夏小洵

发表于 2017-08-23

想起那些清晨
我向妈妈诉说昨晚的梦
都是关于田地，庄稼，牛马，鸡鸭，猫狗

场景和人物开始混乱——活着的和死去的
