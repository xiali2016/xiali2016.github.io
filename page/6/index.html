傍晚的东海城市广场

By 夏小洵

发表于 2016-10-16

一个小男孩坐在地上放声大哭
他的孪生弟兄在安慰他
伸出三个手指头
他和他在讲些什么？

一个小女孩不顾爸爸的反对
爬上已经干涸的水池台阶
她说自己是只水怪
和她一起玩的小朋友也都是水怪

走路蹒跚的奶奶紧紧抓着爷爷的胳膊
爸爸和妈妈边走路边点着手机屏幕
簕杜鹃红艳的花朵已经盛开
有一个婴儿在秋日干燥温煦的风里沉睡

天空中一架波音737轰隆隆飞过
我立即认出了它，带着几分得意

读诗的感想

By 夏小洵

发表于 2016-10-07

　　阅读和翻译露易丝·格吕克的诗歌过程中受到了些启发，有几点感想，写下分享，不一定成熟：

  1. 她欣赏诗的方式：用眼睛听。这又何尝不是她感受大自然的方式？她写大自然时，似乎在倾听大自然的话语——当然是无声的语言；她写诗时也对自然以及万物说出自己的话。

  2. 她在说起自己写诗的时候，有两点对她帮助很大：
    　　其一，是她的教学。她教别人，不是把学生和其他写诗的人当作竞争者，而是她在教他们的过程中，她在热心帮助别人写诗的过程中，她自己也产生了写诗的灵感 —— 关于这点，我自己的理解，是说诗人要拥有爱，心里有了爱，爱帮助别人，爱人类，爱大自然，自己的心灵充满了爱，处于放松的状态，诗也在心中自然产生了。
    　　其二，她分享说：“有时回头看自己写的那些诗，不记得当时怎么写成的，也惊讶自己会写了那么多。有些自己以前写的诗，自己现在读，也读不懂了。” 关于这点，猜想：她写诗时是在充分利用无意识思路流，她应该是经常地、不间断地在写诗，没有停下来（她有段时间患过精神还是心理方面的疾病，厌食症之类的，曾做过一段时间治疗……大概是这样，具体还没有去探究这些），她就是那样不间断地写，无意识的思路流在带着她，她也跟随着那些思路流，这样诗就自然地写出来了。很多作家也都是这样写作的，把日常的写作当成一个习惯，自然就有很多新思想，创新的东西冒出来了。

  3. 看了她的诗，对写乡村方面的诗歌产生了新的思考，有了新的思路。比如可以用这样的思路思考：“用心去倾听，大自然说了些什么，自己又想对大自然说些什么？”　

（2016.10.07）

雪天

By 夏小洵

发表于 2016-10-07

想起了那个雪天
雪花飘落，无声无息
一个小男孩在门口，徘徊

远处，群山模糊
起伏的田野，一片白色
它们是否和天空的灰暗一样
梦中背景单调，色彩暗淡？

门前，没有一丝风
梧桐、香椿、榆树和苦楝子
默默站立，无形的寒冷
见不到那些吵闹的麻雀 ——
它们躲进温暖的窝，悄悄地

暗色的枝桠，亮色的积雪
“咔嚓”，一根小树枝折断了
无声地落回大地
蓬松的雪地接住他，温柔地

他又回到了大地的怀抱
要怎样说呢？
说这世界的负担太沉重
说自己太想家，太想念妈妈？
也许什么都不必说，大地也会懂

(2016.10.07)

October 十月[6]（诗歌翻译）

By 夏小洵

发表于 2016-10-07

作者：露易丝·格吕克 (Louise Glück)
翻译：夏小洵

6.

白天的光明变成
夜晚的光明；
火变成镜子。

我的朋友大地是痛苦的；我想
阳光已经让她失败。
痛苦还是厌倦，很难说清楚。

在她自己和太阳之间，
某些事情已经结束。
她想， 现在， 单独呆会儿；
我觉得我们应该放弃
追问着她去求证确认。

田野的上空，
村庄屋顶的上空，
那些让一切生命成为可能的光芒
化为寒冷的群星

安静躺着，观望：
它们无所给予，也无所求。

从大地
痛苦的耻辱，冷淡和荒芜中

我的朋友月亮升起：
她今晚好漂亮，但她哪时不漂亮？

5.

确实，世界的美不够多。
也确实，我没能力修复它。
那里没有坦诚，这里我也许有点用处。

我在
工作，虽然我一声不吭

平淡乏味的

世界的苦痛
把我们捆绑在一边，小巷

树木成行；我们在这儿是

同伴，都不吱声，
每个人都有自己的思想；

树木后面，私人住宅的
扇扇铁门
百叶窗关闭的房间

不知何故荒凉，废弃了，

仿佛艺术家
有责任去创造

词语本身
虚假，否认
感知的一种手段——交叉路口，

季节的装饰灯。

那时我在这儿还年轻。乘坐
地铁，带着我的一本小书
仿佛它可以保护我自己，抵御

这个相同的世界：

你不是孤身一人，
诗里写道，
在黑暗的隧道中。

October 十月[4]（诗歌翻译）

By 夏小洵

发表于 2016-10-07

作者：露易丝·格吕克 (Louise Glück)
翻译：夏小洵

4.

　

光线变了；
现在中央C音变得暗淡。
晨歌被反复排练。

这是秋天的光，不是春天的光。
秋天的光说：你不会被宽恕。

歌声变了；不可言说
已经进入。
这是秋天的光，不是说
“我已重生”的光。

不是春日黎明：我困顿，受苦，被拯救。
这是现在，无用之物的讽喻。

改变了这么多。你仍然是幸运的：
理想在你体内像发烧那样燃烧。
或者不像发烧，像第二颗心脏。

歌声变了，但仍然动听。
浓聚到更小的空间，心灵空间。
它们此刻是暗色的，凄凉，痛苦。

音符重复。在沉默的期待里
怪异地盘旋。
耳朵习惯了音符。
眼睛习惯了消逝。

你不会得到宽恕，你所爱的也不会得到宽恕。

风吹来吹去，把心灵撕碎；
尾迹里留下奇异的清醒。

你获得了多大的恩典，如此激情地
依恋着你的所爱
希望的丧失也没有摧毁你。

指挥大师，感伤的：

这是秋天的光；我们已被点燃。
当然这是种恩典：直到最后
仍然相信某些事物。

October 十月[3]（诗歌翻译）

By 夏小洵

发表于 2016-10-06

作者：露易丝·格吕克 (Louise Glück)
翻译：夏小洵

3.

雪下过了。我记得
音乐从一扇开启的窗户飘出。

到我这儿来，这世界说。
不是
它真说出了这样的话
而是我感觉这种方式的优雅。

日出。潮湿的薄膜
裹着每一个生命。潭潭冷光
成形于沟渠。

我站立
在门廊里，
多可笑，就像现在看起来这样。

别人在艺术中发现的，
我在大自然中发现了。别人在人类之爱中
发现的，我在大自然中发现了。
非常简单。但那儿寂静无声。

冬天结束了。冰雪融化残留的污尘中，
现出点点绿意。

到我这儿来，这世界说。我那时站立着
穿着羊毛外套，一道明亮的门前 —
我终于可以这样回答
很久以前；这给了我多大的快乐。美
那治疗师，那老师 —

死亡伤害不了我
而你却伤害了我太多，
我心爱的生活。

October 十月[2] （诗歌翻译）

By 夏小洵

发表于 2016-10-05

作者：露易丝·格吕克 (Louise Glück)
翻译：夏小洵

2.

夏天，一个又一个地过去了，
安慰在暴力之后：
说以后会对我好，
现在这对我没什么用了；
暴力已经改变了我。

黎明。矮矮的小山闪光
呈赭色和火红色，田地也闪光
我明白我看到了什么；太阳可能是
八月的太阳，把曾经带走的一切
又还回来了 —

你能听到这声音么？这是我心灵的声音；
现在你不能触碰我的身体。
它已经发生了变化，变得僵硬，
不要再请求它做出回应。

一个如夏日的白天。
特别的安静。枫树长长的树影
落在砾石路面近乎淡紫色。
傍晚，温暖。一个如夏夜的夜晚。

这对我没什么用了；暴力已经改变了我。
我的躯体已经变冷，像清空的田地；
现在只剩下了我的心灵，小心机警地
觉察到自己正被试探。

又一次，太阳如夏日般升起；
慷慨，安慰，在暴力之后。
安慰在树叶变化之后，在田地
收割和翻耕之后。

告诉我这就是未来，
我不会相信你。
告诉我说我正活着，
我不会相信你。

October 十月[1] （诗歌翻译）

By 夏小洵

发表于 2016-10-05

作者：露易丝·格吕克 (Louise Glück)
翻译：夏小洵

《十月》选自诗集《阿弗尔诺》，全首诗共有6小节）

1.

冬天又来了么？天又冷了么？
弗兰克只在冰面跌了一小跤吧？
他的伤好了么？春天的种子播上了么？

夜晚过去了么？
冰融化的水
溢满窄窄的排水沟了么？

我的命
被救下来了吧？安全了么？

伤疤长好了么？疤痕
看不出来吧？

恐怖和寒冷
它们都结束了么？后院的花园
都耙过并栽种上作物了么？

我记得那泥土的感觉，红土，很细密，
那硬硬的田垄上，种子播下了么？
葡萄藤爬上南墙了么？

我听不到你的声音
因为风太吵，呼啸着刮过光秃秃的土地

我已不太在意
它发出什么样的声响了

当我被迫沉默的时候，不知从何时起意识到
去描述那声音没有什么意义

那响声像什么并不能改变它本来是什么

夜晚过去了么？土地耕种后
它就安全了么？

我们播种了么？
土地真的需要我们么？

那些葡萄，都收获了吧？

2016年诺贝尔物理学奖

By 夏小洵

发表于 2016-10-05

三位科学家获得物理学奖
华盛顿大学的索利斯
普林斯顿的霍尔丹
布朗大学的科斯特利茨
他们的伟大贡献：
把拓扑的概念引入物理学研究，
发现物质的拓扑相和拓扑相变。
听起来好高端呀
其实你们小时候都玩过啊
特别是来自农村的小伙伴
玩过泥巴么？
那块泥巴，揉来揉去
不管捏成方形，球形，还是锥形
都在做等价拓扑变换
可别再掏个窟窿，加个环啊
那样又复杂啦
记着不要撕裂或割破
否则就不是拓扑等价啦
大学时学过点流体拓扑学
心说谁没事吃饱撑的研究这玩意儿？
最烦的就是那些点源和点汇
问题都出在它们身上
“喂”，一个常熬夜画网格的哥们问道，
“我到底干的是技术活，还是体力活儿？”
研究拓扑学的数学家听了，笑而不语

（2016.10.05）
