大门外的苦楝子树

By 夏小洵

发表于 2016-11-07

两棵苦楝子并排长在大门外
孩子长，它们也长
苦楝子啊，快成材
老人念叨着，打两套大小柜
给哥哥，给弟弟，摆在新房娶媳妇
圈圈木纹向外荡漾，岁月沉睡，苏醒

两棵苦楝子生长在大门外
春天开满紫色小花，夏天撑出绿色清凉
秋天，叶黄叶落，金黄果子串串
冬天，枝桠秃秃，寒梦陪伴白雪

两棵苦楝子矗立在大门外
一年又一年，猫头鹰在一个黑夜叫声凄厉
喜鹊在一个清晨唱歌报喜
晌午，孩子躺在树阴里的麻袋上酣睡
日落，沉默老人的白发风中凌乱飘起
游子何时回家，何时再次远行？

(2016.11.07)

村南有条小河

By 夏小洵

发表于 2016-11-01

村南有条小河，存在于旧日时光
南坡洼地有泉眼，泉水清清，汩汩冒出
昼夜不息，冬暖夏凉
水草水中招摇，秋蒲占住半个水面
绵鱼从一块石头快速钻进另一块底下
蚂蟥伸缩身体游动，柔软的黑色幽灵
白石板青石板上女人在洗衣说笑
鸭子成群结队，戏水捉鱼，呱呱叫

村南有条小河
大雨过后，河水暴涨，渐渐退去
河套躺满黑白灰褐绿各色的鹅卵石
少年从早晨到傍晚，寻找一块稀奇的红石头
芦苇开始长高，把石头和流水掩藏
草叶间有蚂蚱蹦跶滑翔
低空中蜻蜓盘旋，突然转弯
苇草深处水鸟悄悄筑巢

村南有条小河
花岗岩石条横卧在石墩上
连接着通向学校的田间小道
简易石桥和大自然浑然一体
流水绕过坚硬的石头，哗啦哗啦
一条浑身赤红的鱼逆流而上
背鳍如刀，割开湍急的水流

村南有条小河，流淌在旧日时光
现在都不见了，小河连同北岸的菜地
今天经过这片齐齐整整的梨树园
忍不住对小孩念叨
“我小的时候，这里有条小河……”

(2016.11.01)

月份的名称

By 夏小洵

发表于 2016-11-01

看到英文的十二个月
January, February, March
很个性，有感觉
个性化的月份名字只存在古代农历中
每个月的称呼有多种说法

一月，二月，三月
就像给孩子起名大毛、二毛和三毛
无内涵，无美感
现代人坐在一尘不染的办公室
有空调，有日光灯
工作化为股股数字流
不再耕种，不再沾染泥土
用消除歧义的数字记事
一切井井有条，一切准确无误

古代农家耕作劳苦，文人有点浪漫
用农历，用个性名字，用文字记事
正月，仲春，暮春
初夏，仲夏，伏月
初秋，仲秋，深秋
初冬，冬月，腊月
窗外的风变冷了，已是初冬

(2016.11.01)

忘却和怀念被赋予了什么意义？
痛苦和快乐沉湎于相互替代的游戏。

《小森林》之夏秋冬春篇

By 夏小洵

发表于 2016-10-27

《小森林》是日本的一部电影，分成夏秋冬春篇四个片段，正好四季合成完整的一年。我觉得它拍摄得更像是一部纪录片，类似于《舌尖上的中国》之类的记录片。电影主要讲述了一个在城里混得不顺利的年轻女孩市子，回到日本东北部的一个偏远的农村老家，一年中发生的事情，或者说是她一年生活中的点点滴滴的记录。影片的拍摄地点是日本岩手县奥州巿农川区上大森。

故事发生的地方是偏僻的村落一角，是一个小盆地，叫小森，那里有市子的家，她家的田，是她长大的地方，也是妈妈和她共同生活过的地方。而现在，妈妈已经不知道去了哪里——妈妈在5年前突然不辞而别，杳无音信。离家出走的孩子很多，离家出走的妈妈很少，不是么？

日本的电影，日本人的生活，即使是日本农村人的生活，都蕴涵着他们的民族特性——精致，这是看电影时的感觉。我也是在渤海边的农村长大的，至少在地理和天气上，和日本的农村有不少相似处，所以看完有一些感触。这部电影在豆瓣的得分非常高，目前是8.9分（我发现这分数也是在变化的，我看的时候是9.0分），这样的分数，应该可以把它划分到优秀电影里去了。

看完电影后写下这些，不算是影评，只是些随想吧。

不一样的农村

故事发生在日本东北部的偏远的农村，市子家是栋独门独户的二层小楼。

日本毕竟属于发达资本主义国家，有人这么说过：

看一个国家是否发达不是去比那些大城市，而是要去看那些偏远的农村。

确实中国的很多大城市，如北上广深，其发达程度堪比任何发达国家的大城市。然而在农村呢？发达国家和发展中国家的差别立刻就显现出来了。不说别的什么地区，我小时候生活过的胶东半岛的农村，在中国算不上太富裕的地方，但也绝不能算是贫困地区，跟影片里的日本的“偏僻农村”相比，与人家还是有差距的。

市子房子里的家居简约而精致，特别是厨房里的那些锅碗瓢盆，看着就舒服，屋里还有只很有性格的猫。记得我妈来南方住的几年，她很迷信日本的产品，说日本产质量好。其实，妈妈对日本人，从感情上是很反感的，她也念叨说日本人坏，可是这并不影响她喜欢日本的产品。她回老家前，专门从无印良品店买了两把日本产的不锈钢削皮器，说那东西很锋利，怎么用也不会钝，削土豆皮、萝卜皮、苹果皮，好使得很。她有时也抱怨这城市里的水果太贵，总拿这里的价格和老家的价格对比，她自己很少吃水果，舍不得。但是要买给孙子吃，即使是山姆会员店里的那些进口的车厘子，她花起钱来也毫不犹豫。妈妈没文化，但是运用着自己朴素的人生经验和哲理生活着。好东西，人们总是喜欢的，不论是中国产，还是日本产、美国产。

日本的产品虽好，价格当然也比中国产贵很多，但我妈认为这很值——她的很多生活理念一点也不像一个没文化的农村人，甚至比我都超前，比如她喜欢在能承受的范围内追求高质量和高品质，特别是那些她认为重要的东西，绝不去凑合，宁可在别的方面省点，也要追求好的。其实我没有跟她说，在日本大米和蔬菜等的价格更贵，日本的很多农产品都贵得很，这也是很多胶东蔬菜出口日本很具竞争力的原因。

农产品的价格，比如玉米小麦，是国家定价的，非常便宜，种一亩地，除去农药化肥的钱，赚不了几百元——乡村怎么能不衰落呢？当然美国和加拿大的农产品也便宜，他们地多人少，大规模种植，靠量取胜，又有政府的补贴，人家的农民叫农场主，生活质量一点不比城市人差。

不在中国农村生活过的人，可能不理解农村人在这个国家所承受的苦难，从人民公社大集体时自己吃不饱饭也要赶着马车交公粮，到实行生产责任制时人们起早摸黑在田里劳累换得连年大丰收的喜悦，到乡镇企业的兴起纷纷进厂做工，到城镇化去大城市和小城市暂住，再到自己生活的土地被莫名奇妙地划归到城市版图，成为郊区，得到了当时看似挺多的一笔钱，永远失去了土地，成为了无根的植物。

现在中国很多地方的乡村在衰败，这是个不争的事实：人口老化，田地荒芜，老人和儿童在留守，村里”有出息“的年轻人都去了大城市，在那里努力拼搏着梦想着扎下根，农村也是各种假冒伪劣产品的倾销地，环境被污染，很多小河干涸了或者河水变了颜色，如果当地发现了什么矿产，那可能真不是件好事——当然以上这些可能更侧重于现在农村阴暗的一面，略显片面。 中国也确实有很多风景美如画的农村，有些农村也相当富裕。

确实看这部电影时，联想了很多有关中国的农村。记得有人在网上嘲笑说，很多上海女人崇洋媚外，喜欢嫁到日本的农村，好像是大城市嫁到农村去受罪。在农村和大城市都曾生活过，与其在香港上环的那些阴暗狭小的唐楼房间里憋屈地住着，真不如在日本那样的农村住着舒服呢，住在那样的农村简直是住在别墅里过着土豪的田园生活。无论哪里的普通人，都向往和追求更好的生活吧。

当然以后中国农村的生活也会越来越好，希望是这样。如果我们的农村赶不上那些发达国家，就是那些一线城市再发达，在我眼里也称不上一个发达国家。更为可悲的是，很多大城市的发展，是靠抽周边不发达地区的血而形成的，各种购房限制政策，在城市打工交了各种社保，离开城市要回老家取钱时被七扣八扣，自己交的钱自己说了不算。在大城市里脏乱危险的活儿，总是用那些临时工去做，至于那些易得职业病的工作更不用说了。

《小森林》里的日本农村其实也面临着一些问题，比如年轻人去城里工作，人口严重老龄化，老龄化其实不光是农村，也是整个日本现在面临的问题。但也看到，这里的乡村环境保护得很好，景色优美，小河的流水很清澈，偶尔可以见到条逆流而上的鳟鱼，淡水里生长的河蟹爬到农田里，山林保持着原始的生态，各种野花，各种树，各种果子，那些鸟儿，那些小动物如松鼠等，甚至还要担心熊出没。

　　

不如归去：种田，做菜，看书，看星星，睡觉…

一个在城市生活得不太顺利的年轻女孩市子，回到日本东北部的一个偏远的农村老家，在小森林那里度过了看似孤独实则心灵安静的一年，她都经历了什么呢？

电影里讲的是有关家乡、童年、朋友、同学，家常美食和妈妈的故事。

这部电影其实没有什么故事情节，之所以得到了好评，引发了观众心里的共鸣，肯定是胜在其他方面，比如那些美丽的乡村景色，那些独处时的安静，当人没有其他人说话时，才会听得到自己心中的絮絮叨叨。儿时的朋友，那些大妈大叔，那些家常菜，那些食物味道的细节会让人回味起儿童时光，妈妈总是和儿时的美食联系在一起的，想到妈妈还会想到些什么？

农村的家乡

在《论语·先进》篇有这样一段：

… 子曰：“何伤乎？亦各言其志也。” 曰：“莫春者，春服既成，冠者五六人，童子六七人，浴乎沂，风乎舞雩，咏而归。” 夫子喟然叹曰：“吾与点也！” …

今译：

… 孔子说：“那又有什么妨碍呢？也就是各人谈谈自己的志向啊！” [曾点]说：“暮春时节，春天的夹服已经穿定了，和成年人五六人，少年六七人，去沂河洗洗澡，到舞雩台上吹吹风，唱着歌一路走回来。” 孔子长叹了一声，说：“我是赞成曾点的。” …

对《论语》的解读，是门高深的学问，也是最容易被统治阶级所利用曲解的，我的曲解，这段话好像现在那些北上广深大城市中打拼的年轻人聊天时的话语：

一个说：“谈谈理想的生活吧！”，另一个说：“好想回老家的农村去啊，没这么大的工作压力，过着悠闲的乡村生活。” 这人接过话：“哎呀，我也有这样的想法啊！”

《小森林》这部电影就是要满足一个城市人的这种梦想。作为一个在农村生活过的人，不想说太多什么，这时只想充满正能量，跟着电影的镜头去细细体味那些美好的乡村生活。

还有乡村里和谐的邻里关系，和几个爱聊天的大妈或者和儿时的好友泡上一壶茶，边喝边聊……

那些菜谱

稻田拔草累啦，收稻子累了，去打零工搬运鳟鱼累了，不要紧，回家做道家常美食，犒劳一下自己，补一补吧。

《小森林》27道诱人美食详解教程（超长版，巨详细）

第28道菜是芥末白菜，哈哈，这是我自己做的一道开胃小菜。

那一道道的美食是这部电影的重头戏，也是电影的灵魂，美味把一切联系在一起。有人把《小森林》称为“舌尖上的日本”，确实是有道理的。

妈妈

对女孩来说，妈妈既是一种温暖的童年回忆，也掺杂着太多的其他情结，就像一个儿子长大以后要么成为他的父亲，要么成为他的父亲的反面。女儿和妈妈的关系，也会是这样么？隐隐感觉妈妈是为了追求自己的幸福，去找她心爱的男人去了，或者过去没有那种勇气，但是五年前的某一天，她终于下定了决心，最终离家出走。后来妈妈也给女儿市子来信了，不是么？在冬天的一个寒冷的一天，地上的积雪还很厚。

其实还有一些大大的问号：

市子的爸爸是谁？难道她是在单亲家庭里长大的么？

如果市子那时是20岁，5年前市子才15岁啊，妈妈离开后她一个人是怎么过的？

儿时朋友

儿时的朋友，不管是男是女，那些友谊都是融在血液里，融在熟悉的乡音里。就像我们都曾在村西的那条河里捉过鱼，都曾爬上村东的那棵大树上摘过果子。

女主角

女主角市子由 桥本爱 饰演，是个清新的美女啊，好友纪子由 松冈茉优 饰演，也是个美女。

不说电影，现实生活中，现在农村的孩子也挺娇生惯养的，生活不错，出个美女其实也平常，但再深入点，要论论思想和教养之类的，就……呵呵

漫画

这电影是根据漫画家 五十岚大介 的原作改编的。

啰啰嗦嗦写这么多，其实你有看这些废话的功夫，不如自己去看看电影啦！

(全文完)

扁豆

By 夏小洵

发表于 2016-10-26

园子里矮矮的石墙爬满了扁豆
绿色蔓藤的梢头开着
白色的，黄色的，紫色的花
它们在微风中不断点着头
那样会引来美丽轻巧的蝴蝶——
黑色的带着黄色亮斑
白色的杂着黑色圆点
黄色的镶着暗色纹线
还会有蛾子到来——
不过要等到傍晚
它们又胖又笨，到处乱撞

墙根是黑色的松软泥土
盖满了快要腐烂的黑色麦秸
那土下一定有蚯蚓在慢慢爬动
它们为什么那么喜欢黑暗，喜欢泥土？
那土中也一定有白白的胖虫子
它们都是些小坏蛋
正在悄悄噬咬某棵扁豆的嫩根

那些扁豆已经结荚

诗

读诗的感想

By 夏小洵

发表于 2016-10-07

　　阅读和翻译露易丝·格吕克的诗歌过程中受到了些启发，有几点感想，写下分享，不一定成熟：

  1. 她欣赏诗的方式：用眼睛听。这又何尝不是她感受大自然的方式？她写大自然时，似乎在倾听大自然的话语——当然是无声的语言；她写诗时也对自然以及万物说出自己的话。

  2. 她在说起自己写诗的时候，有两点对她帮助很大：
    　　其一，是她的教学。她教别人，不是把学生和其他写诗的人当作竞争者，而是她在教他们的过程中，她在热心帮助别人写诗的过程中，她自己也产生了写诗的灵感 —— 关于这点，我自己的理解，是说诗人要拥有爱，心里有了爱，爱帮助别人，爱人类，爱大自然，自己的心灵充满了爱，处于放松的状态，诗也在心中自然产生了。
    　　其二，她分享说：“有时回头看自己写的那些诗，不记得当时怎么写成的，也惊讶自己会写了那么多。有些自己以前写的诗，自己现在读，也读不懂了。” 关于这点，猜想：她写诗时是在充分利用无意识思路流，她应该是经常地、不间断地在写诗，没有停下来（她有段时间患过精神还是心理方面的疾病，厌食症之类的，曾做过一段时间治疗……大概是这样，具体还没有去探究这些），她就是那样不间断地写，无意识的思路流在带着她，她也跟随着那些思路流，这样诗就自然地写出来了。很多作家也都是这样写作的，把日常的写作当成一个习惯，自然就有很多新思想，创新的东西冒出来了。

  3. 看了她的诗，对写乡村方面的诗歌产生了新的思考，有了新的思路。比如可以用这样的思路思考：“用心去倾听，大自然说了些什么，自己又想对大自然说些什么？”　

（2016.10.07）

雪天

By 夏小洵

发表于 2016-10-07

想起了那个雪天
雪花飘落，无声无息
一个小男孩在门口，徘徊

远处，群山模糊
起伏的田野，一片白色
它们是否和天空的灰暗一样
梦中背景单调，色彩暗淡？

门前，没有一丝风
梧桐、香椿、榆树和苦楝子
默默站立，无形的寒冷
见不到那些吵闹的麻雀 ——
它们躲进温暖的窝，悄悄地

暗色的枝桠，亮色的积雪
“咔嚓”，一根小树枝折断了
无声地落回大地
蓬松的雪地接住他，温柔地

他又回到了大地的怀抱
要怎样说呢？
说这世界的负担太沉重
说自己太想家，太想念妈妈？
也许什么都不必说，大地也会懂

(2016.10.07)
